# FundMatch

Quadratic funding platform for public goods.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/FundMatch.sol`
- Stacks: `contracts/stacks/contracts/fundmatch.clar`

## Setup
```bash
pnpm install
pnpm dev
```
